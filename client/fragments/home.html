Chatroom is being shut down in June 2024

Starting June 11th 2024, we will discontinue and shut down all the Chatroom services. If you have important data or messages you need to export or backup, please use the corresponding Chatroom features before June 11th to ensure you keep access to your data. Equestria.dev makes no guarantee whatsoever that data will remain accessible past this date.

Chatroom is now released under the GNU AGPLv3 license. Equestria.dev has recently announced that it would be releasing all of its software under the GNU AGPLv3 license instead of the MIT license, to further tighten control over actors making use of the source code. This might affect your use of Chatroom if you are running a modified version.

Chatroom 2.8.3 (Long Term Support)

Chatroom 3 (Codename "Heeler") is now under development (it can be enabled with a few secret options)! While regular Chatroom features will continue to function normally, select people will be able to test new features. Contact your administrator for details. This does mean that Chatroom 2.8 is the last version of Chatroom 2 before the release of Chatroom 3. This old version of Chatroom will only contain bug fixes and no new features will be added.

  • 2.8.3 Fix: Chatroom no longer gets stuck on connecting with the Spyglass server.
  • 2.8.2 Fix: Chatroom no longer uses a lot of CPU resources and creates a lot of processes (about 60).
  • 2.8.1 Fix: Links can now open in new tabs if indicated by the website or if you hold down Cmd/Ctrl while clicking it.
  • 2.8.0 Fix: Clicking on the search bar on the home page now focuses the search bar on top.
  • 2.8.0 Feature: Introducing Chatroom Spyglass: a completely transparent utility to collect system information and help improve Chatroom.
    • Chatroom automatically detects if your device is eligible to Spyglass. If it is making a mistake, please contact your administrator.
    • Users with the "forcespyglass" experiment set to "Enabled" will always have Spyglass enabled.
    • Contact your administrator for details regarding what data is collected and how this data is used.
  • 2.8.0 Fix: The Chatroom3PID API is now deprecated and will be removed in Chatroom 3.

    • WARNING: Make sure you only use third-party authentication WITH TRUSTED WEBSITES as this gives the website full access to your account.
